## WEEKLY SCRIPTS

These describe the scenes for this week's content. Use them to identify which
scene this image belongs to and insert a brief scene label AFTER the layer
prefix (UpperCamelCase, English only, letters+digits).

- Translate scene names to English even if the script is in another language.
- Format: Layer + Scene + rest.
- Group sub-scenes by meaning, not by scene number.
- One scene may have multiple angles or variants - keep the scene name
  identical across all of them and append a short visual descriptor for what
  differs: angle, composition, moment, or beat.

Examples:

- same scene, different angles -> "BgCakeDinnerWide", "BgCakeDinnerClose",
  "BgCakeDinnerBehind"
- same scene, different moment -> "BgCakeDinnerArrival", "BgCakeDinnerPanic"
- foreground variants -> "FgCakeDinnerMirandaCold", "FgCakeDinnerAndyShocked"

If the image does not clearly match any scene, omit the scene label.
