minion canvas — React Flow placeholder

This container is a stand-in for the visual canvas. It will call the atomic services over the web, exactly like n8n does — one node per service, wired on a graph.

The services are already live in this same project. Try one directly:

curl -F file=@photo.jpg http://localhost:8091/run-file --output blurred.jpg

Each service also serves /healthz and /openapi.json. Swap this mounted folder (deploy/reactflow/) for the built canvas app when it exists.
